## What is AniCircle?

**AniCircle** is a modern, feature-rich anime tracking platform designed for passionate anime enthusiasts. Built with cutting-edge technologies, it provides a seamless experience to manage your anime journey.

Whether you're a casual viewer or a hardcore otaku, AniCircle helps you:

- **Track** every episode you've watched
- **Rate** and review your favorite series
- **Organize** your watchlist efficiently
- **Connect** with friends and share recommendations
- **Discover** new anime through MyAnimeList integration

## Quick Start

<img src="https://media.giphy.com/media/WUlplcMpOCEmTGBtBW/giphy.gif" width="250" alt="Coding"/>

</div>

<br/>

### Prerequisites

- Node.js 18+
- Supabase account
- (Optional) Google Cloud Console for OAuth

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/anicircle.git

# Navigate to project
cd anicircle

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
```

### Environment Variables

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
SUPABASE_SERVICE_ROLE_KEY=your_service_role_key
SUPABASE_DATABASE_URL=your_database_url
```

### Run Development Server

```bash
npm run dev
```

The app will be available at `http://localhost:5000`

## API Endpoints

</div>

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/api/profile` | Get current user profile |
| `GET` | `/api/anime` | Get user's anime list |
| `POST` | `/api/anime` | Add anime to list |
| `PATCH` | `/api/anime/:id` | Update anime entry |
| `DELETE` | `/api/anime/:id` | Remove anime from list |
| `GET` | `/api/friends` | Get friends list |
| `POST` | `/api/friends` | Send friend request |
| `GET` | `/api/notifications` | Get notifications |
